### 目录结构
```json
--server
    |-- api: controller
    |-- cmd: 通用命令
    |-- conf: 配置文件，以及初始化
    |-- model: 数据模型
    |-- dao: 数据访问
    |-- router: 路由
    |-- service: 业务逻辑
        |-- dto: 数据传输对象
    |-- utils: 工具类
    |-- global: 全局变量
        |-- constants: 全局常量
    |-- log: 日志
    |-- middleware: 中间件

```
